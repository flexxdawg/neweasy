# 🚀 Tech Tutorials Platform

A modern and animated tech tutorial website built with **HTML, CSS and JavaScript**, featuring a futuristic UI inspired by video game interfaces.

## 🎮 Features

* 🖥️ Futuristic / cyberpunk design (blue, black, white)
* ✨ Smooth animations using GSAP
* 🎯 3D hover effects on cards (Vanilla Tilt)
* 🌌 Animated 3D background (Three.js)
* ⏰ Live clock display
* 🌐 Public IP detection
* 📚 Organized tutorial categories:

  * DevOps
  * System & Network
  * Cybersecurity
